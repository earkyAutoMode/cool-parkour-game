# Cool Parkour Game (炫酷跑酷小游戏)

基于 HTML5 Canvas 开发的炫酷赛博朋克风格跑酷小游戏。

## 游戏特点
- **核心玩法**：角色自动向右奔跑，玩家通过点击屏幕或空格键控制角色跳跃。支持 **二段跳**。
- **视觉风格**：霓虹/赛博朋克 2D 风格，具有多层背景滚动（Parallax Scrolling）效果。
- **物理引擎**：实现重力、跳跃冲量以及与障碍物的碰撞检测。
- **动态生成**：障碍物（箱子、尖刺等）随机从右侧生成。
- **计分系统**：根据奔跑距离实时计分，记录本次最高分。
- **粒子特效**：跳跃、落地和碰撞时伴有发光粒子效果。

## 操作说明
- **跳跃/开始游戏**：空格键 (Space) 或 点击屏幕。
- **二段跳**：在空中再次按下跳跃键。

## 技术栈
- 纯 HTML5 / CSS3 / JavaScript (原生)。
- Canvas API 绘图。
- 响应式设计。
